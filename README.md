# 🎓 E-Learning Personalization Platform

An intelligent web-based learning system that delivers personalized educational content based on user interests and performance.

Developed as part of academic project work.

---

## 📌 Project Overview

The **E-Learning Personalization Platform (E-LPP)** is a full-stack web application designed to provide customized learning experiences for students.

The system supports:

- Secure user authentication
- Course management
- Quiz and assessment system
- Progress tracking
- Role-based access control
- Personalized course recommendations

---


- Frontend handles UI & interaction
- Backend processes business logic
- MongoDB stores all data
